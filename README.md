# Midnight Cheat Sheet

A quick-reference addon for **World of Warcraft: Midnight** (12.0.1 Season 1) by **Chûd-Ravencrest** (EU).

Stat priorities, enchants, gems, consumables, BiS wishlists, and gearing info — all in one panel, no alt-tabbing required.

## Features

**Stats** — Per-hero-tree stat priorities with live combat ratings and DR bracket indicators.

**Consumes & Enchants** — Best-in-slot enchants, gems, flasks, food, potions, oils, and runes for your spec.

**Wishlist** — Track the gear you're chasing across multiple lists. Comes with preset BiS lists for every spec (blue border), plus your own custom lists (green border).

**Gearing** — Season 1 gearing tracks at a glance: M+ crests, raid drops, crafted gear, delves, and Dawncrest vendors.

All **13 classes** and **40 specs** supported, including the Midnight-exclusive Demon Hunter: Devourer.

## Wishlist Highlights

- Create multiple named lists per spec (e.g. "Raid ST", "M+", "Delves")
- **Alt-click** any item in the Adventure Guide, chat links, or the addon itself to add/remove from wishlists
- Wishlisted items are flagged in tooltips game-wide
- Cross-spec management — wishlist items for any spec from any spec

## Installation

1. Download and extract into `World of Warcraft/_retail_/Interface/AddOns/`
2. `/reload` or restart the client
3. `/mcs` to open

## Slash Commands

| Command | Action |
|---|---|
| `/mcs` | Toggle the panel |
| `/mcs wishlist` | Print current wishlist to chat |
| `/mcs reset` | Reset all saved data |
